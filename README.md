# Reply Later

> Not a to-do list. A place to drop thoughts—and have the right ones come back when it matters.



## What this is (AHDD maybe?)

This came from a simple problem.

Too many small thoughts, all at once.
None urgent, but all distracting.

“Email this.”
“Look that up.”
“Random idea.”
“Don’t forget…”

Most tools make you deal with them immediately—organize, prioritize, decide.

I didn’t want that.

So this is a place where you can just **put the thought down and move on**.



## How it works

Type something.
Press enter.
It disappears.

Later, it might come back.

Just one thing at a time.

You can:

* handle it
* ignore it
* let it go

If it doesn’t matter, it fades out.



## Why this exists

Not everything needs to be tracked.

Sometimes you just need a place to **offload a thought** without carrying it around.

This is less about productivity, more about **mental space**.



## What it’s not

* not a to-do list
* not a notes app
* not something you maintain

You don’t manage this.



## Status

Very early. Just testing the idea.
